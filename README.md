//Calculadora cientifica "Legión de exploración"
Proyecto Swing en JavaNeatbens desarrollado con metodologia Scrum. Este cuenta con una infrestuctura que incluye 
funcionalidades basicas y cientificas (Suma, resta, división, multiplicación, potencia, reiz cuadrada, memoria, etc).

//Requisitos
Para poder hacer uso de la calculadora se debe contar con el entorno java.
Para ello se debe contar con los siguientes requisitos: 
- JAVA JDK 20 o superir
- Apache NetBeans IDE 24
- 1 MB de almacenamiento disponible!!!
- Tener acceso a GitHub

//Clonar el proyeccto
Para clonar el repositorio se debe primero abrir JAVA NEATBEANS.
- Direccionandonos a Menú: Team - Git - Clone
- Pegar la URL del repositorio de la calculadora
-Seguir el asistente hasta finalizir 
-Abrir el proyecto clonado en NeatBeans

//Ejecutar la calculadora
-En NetBeans se debe hacer click derecho encima del proyecto "Legion Explor" dando a Clean and Buaild
-De nuevo procedemos a dar click derecho y ejecutar "Run"
-Probar las duncionalidades

//Funcionalidades actuales del MVP
Dentro de la calculadora se encuentras las funcionalidades trigronometricas inversas, trigronometricas, funciones
de memoria, operaciones avanzadas y constantes, funciones logarítmicas y exponenciales, operaciones básicas y 
paréntesis, funciones de limpieza y números y funciones básicas. Esta cuenta con una interfaz grafica la cual cuenta 
con una espesaio para mostrar operaciones y botones que estan agrupadas y clasificas con ayuda visual. 

1.Funciones trigonométricas inversas (Amarillo).
arcsin - Arcoseno: Devuelve el ángulo en radianes cuyo seno corresponde al número dado.
arccos - Arcocoseno: Devuelve el ángulo en radianes cuyo coseno corresponde al número dado.
arctan - Arcotangente: Devuelve el ángulo en radianes cuya tangente corresponde al número dado.

2. Funciones trigonométricas (Naranja).
sin - Seno: Calcula el seno del valor ingresado en radianes.
cos - Coseno: Calcula el coseno del valor ingresado en radianes.
tan - Tangente:  Calcula la tangente del valor ingresado en radianes.
cot - Cotangente: Calcula la cotangente del valor ingresado en radianes.
sec - Secante: Calcula la secante del valor ingresado en radianes.
csc - Cosecante: Calcula la cosecante del valor ingresado en radianes.

3. Funciones de memoria (Azul Claro).
MS - Memory Store (Guardar en memoria): Guarda el número actual en memoria.
MR - Memory Recall (Recuperar de memoria): Recupera el número guardado en memoria.
MC - Memory Clear (Limpiar memoria): Borra el valor guardado en memoria.
M+ - Memory Add (Sumar a memoria): Suma el número actual al valor en memoria.
M- - Memory Subtract (Restar de memoria):Resta el número actual al valor en memoria.

4. Operaciones avanzadas y constantes (Azul).
|x| - Valor absoluto: Calcula el valor absoluto.
ⁿ√ - Raíz n-ésima: Calcula la raíz n-ésima del número.
√ - Raíz cuadrada: Calcula la raíz cuadrada.
ʌ - Potencia: Eleva un número a una potencia.
π - Constante Pi: Inserta el valor de pi.

5. Funciones logarítmicas y exponenciales (Morado).
xⁿ - Potencia específica: Eleva un número a una potencia específica.
eˣ - Exponencial (e elevado a x): Calcula e elevado al número.
log - Logaritmo base 10: Calcula el logaritmo base 10.
ln - Logaritmo natural: Calcula el logaritmo natural (base e).

6. Operaciones básicas y paréntesis (Rojo).
( - Paréntesis izquierdo:  Abre un paréntesis.
) - Paréntesis derecho: Cierra un paréntesis.
% - Porcentaje: Calcula porcentaje.
÷ - División: Realiza división.
× - Multiplicación: Realiza multiplicación.
- - Resta: Realiza resta.
+ - Suma: Realiza suma.

7. Funciones de limpieza (Verde).
CE - Clear Entry (Borrar entrada): Borra solo la entrada actual.
C - Clear (Borrar todo): Borra todo el contenido.

8. Números y funciones básicas (Grises).
0-9 - Dígitos numéricos: Permiten ingresar números.
. - Punto decimal: Agrega punto decimal.
+/- - Cambiar signo (positivo/negativo):  Cambia el signo del número.
= - Igual (calcular resultado): Muestra el resultado de la operación.
